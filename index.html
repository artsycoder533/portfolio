Talk2Do

Talk2Do is a single page notes application that allows a user to operate its functionality without typing by utilizing the Web Speech API. For added convenience, the notes are saved to local storage via the Web Storage API. Since the Web Speech API is still in its experimental phase, there is limited browser compability.

Built With: HTML5, CSS3, Javascript, Web Speech API, and Web Storage API
